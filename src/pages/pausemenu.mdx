# Nil's Pausemenu

A modern and customizable pause menu replacement for FiveM servers, providing a sleek interface with various features.

## Purchase

You can get this script from the [ERR Tebex-store](https://err.tebex.io/package/6685111)

## Installation

<Callout type="info">
Make sure to follow the installation instructions carefully to ensure proper functionality.
</Callout>

<Steps>
### Install dependencies
- [ox_lib](https://github.com/overextended/ox_lib)

### Add to Server Resources
- Extract the nil-pausemenu folder from the downloads folder
- Place the nil-pausemenu folder into a new [ERR] directory within your server's resources folder

### Update Server Configuration
Open your server.cfg file and add one of the following lines:

<Tabs items={['ensure', 'start']}>
  <Tab>
  ```cfg
  ensure [ERR]
  ```
  </Tab>
  <Tab>
  ```cfg
  start [ERR]
  ```
  </Tab>
</Tabs>
</Steps>
