# Exports

`err_quickmenu` provides exports to dynamically manage radial menu options.

## addOption

Adds one or more options to the radial menu.

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    local id = exports['err_quickmenu']:addOption({
        id = 'garage',
        label = 'Garage',
        icon = 'warehouse',
        onSelect = function() print('Garage') end
    })
    ```
  </Tab>
  <Tab>
    ```ts
    import { exports } from '@resource_name';

    let id = exports['err_quickmenu'].addOption({
        id: 'garage',
        label: 'Garage',
        icon: 'warehouse',
        onSelect: () => console.log('Garage')
    })
    ```
  </Tab>
</Tabs>

## removeOption

Removes a menu option by ID.

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    exports['err_quickmenu']:removeOption('garage')
    ```
  </Tab>
  <Tab>
    ```ts
    exports['err_quickmenu'].removeOption('garage')
    ```
  </Tab>
</Tabs>

## registerSubItems

Adds sub-items to an existing parent menu.

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    exports['err_quickmenu']:registerSubItems('garage', {
        { id='wash', label='Car Wash', icon='soap', onSelect=function() print('Wash') end }
    })
    ```
  </Tab>
  <Tab>
    ```ts
    exports['err_quickmenu'].registerSubItems('garage', [
      { id:'wash', label:'Car Wash', icon:'soap', onSelect:() => console.log('Wash') }
    ])
    ```
  </Tab>
</Tabs>

## clearRadialItems

<Tabs items={["Lua", "JS"]}>
  <Tab>

  </Tab>
  <Tab>

  </Tab>

  </Tab>
  <Tab>

  </Tab>

## getRadialMenuState

Returns the current menu state.

<Tabs items={["Lua", "JS"]}>
  <Tab>
    ```lua
    local state = exports['err_quickmenu']:getRadialMenuState()
    print(state.open, state.disabled)
    ```
  </Tab>
  <Tab>
    ```ts
    const state = exports['err_quickmenu'].getRadialMenuState()
    console.log(state.open, state.disabled)
    ```
  </Tab>
</Tabs>
